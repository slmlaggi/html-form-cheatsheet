HTML Form Reference

Form Tags

Tag Name Description Notes
<form> Form The wrapper element for all HTML forms. The form element can contain one or more of the mentioned form elements below. Attributes
<input> Input Specifies an input field where the user can enter data. The input element can be displayed in several ways, modified by the type attribute. (Defauls to "text" if not specified.) Attributes
<textarea> Text area Defines a multi-link text input control. Often used to collect user inputs like comments or reviews. Attributes
<button> Button Defines a clickable button Text can be put inside a button element. Always specify the type attribute for a button element. Attributes
<label> Label Defines a label for elements such as <input> and <textarea>. Using labels with the elements listed would benefit the accessibility of them. Attributes
<select> Select Creates a drop-down list. The <option> tags within the select element define the available options in the drop-down list. Attributes
<option> Option Defines an option in a select list. Goes inside a <select> or <optgroup> element. Can be used without any attributes, but the value attribute is usually needed.
Related options can be grouped within the <optgroup> tag for longer lists of options. Attributes
